CodeBuddy is a full-stack AI-powered web application designed to help learners understand code at different skill levels,from beginner-friendly (“baby”) explanations to expert-level breakdowns.

- Features
Multi-level explanations (Baby, Beginner, Intermediate, Expert)
Line-by-line code explanations
Concept extraction from code
Auto-generated quizzes for learning reinforcement


- Tech Stack
Frontend: React, Tailwind CSS
Backend: Node.js, Express
Database: MySQL (+ in-memory fallback)
AI Integration: Optional API integration (Anthropic/OpenAI-style)


- Use Case
 Designed for students, beginners, and developers who want to understand code deeply and interactively.
